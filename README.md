About
A simple application to help you keep track of your tasks. Create different projects and add todos to each one. All data is saved locally in your browser.

Features
Create multiple projects to organize your tasks

Set priority levels (Low, Medium, High)

Add due dates to tasks

Mark tasks as complete

Edit or delete tasks

Data persists after closing the browser

No account or internet connection required

How to Use
Online
Visit the live demo at the link above.

Locally
Download all three files (index.html, style.css, script.js)

Place them in the same folder on your computer

Double-click index.html to open in your browser

Quick Guide
Action	How to

Add a todo	Click "+ Add Todo" and fill in the form
Edit a todo	Click on any todo to modify it
Mark complete	Check the box next to the todo

How Data is Stored
All your todos are saved using your browser's localStorage. This means:

Your data stays on your computer

It remains when you close and reopen the browser

It is private and not uploaded to any server

It will not sync across different devices or browsers
